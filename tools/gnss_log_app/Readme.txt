How to Use
1. Turn on your device and SP BT
2. Launch gnss_log APK and Click "start" button to scan BT devices
3. Select your device to connect BT SPP Server [uuid: 0000EE01-0000-1000-8000-00805F9B34FB]
4. Receive GNSS Debug Log via BT SPP
5. Click "Stop" button and show Log file path
