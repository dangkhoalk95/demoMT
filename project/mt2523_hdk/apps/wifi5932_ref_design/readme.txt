@par Overview
  - Example project description
    - This project demonstrates the Wi-Fi scan 
      features of the LinkIt 5932 HDK through the following:
      - How to use the host device MT2523 to initialize the Wi-Fi.
  - Application features
    - Act as a Wi-Fi station to scan the Access Points.

@par Hardware and software environment
  - Supported platform
    - MediaTek LinkIt 5932 HDK.
  - MT5932 HDK Jumper configuration
    - Remove Jumper from J2201 to disable UART download mode
    - Place Jumper on J2204(H) for Host Interface Enable
  - Environment configuration
    - A serial tool is required, such as hyper terminal or TeraTerm for UART
      logging.
    - COM port settings. baudrate: 115200, data bits: 8, stop bit: 1, parity:
      none and flow control: off.
    - UART logging port
        - MT2523 UART0 for debug log
        - MT2523 UART on MK20 USB for AT Command RX/TX
        - MT5932 UART (J2103 G19/G20) for debug log
  - Pin connection between MT5932 HDK and MT2523 HDK
       +---------------+-----------------------+
       | MT2523 HDK    |       MT5932 HDK      |
       +---------------+-----------------------+
       |  J1001(2)     |       J2103(GPIO13)   |
       +---------------+-----------------------+
       |  J1002(2)     |       J2103(GPIO14)   |
       +---------------+-----------------------+
       |  J1003(2)     |       J2103(GPIO15)   |
       +---------------+-----------------------+
       |  J1004(2)     |       J2103(GPIO16)   |
       +---------------+-----------------------+
       |  J1005(2)     |       J2103(GPIO11)   |
       +---------------+-----------------------+
       |  J1006(2)     |       J2103(GPIO12)   |
       +---------------+-----------------------+
       |  Sensor(19)   |       J2101(GPIO4)    |
       +---------------+-----------------------+
       |  Sensor(5)    |       GND             |
       +---------------+-----------------------+
       |  Sensor(13)   |       J2101(2)        |
       +---------------+-----------------------+
       Table 1. Pin connection
      To run the project on the LinkIt 5932 HDK:
       - Connect MT5932 to MT2523 according to table 1.
       - In this mode, if the power is on, MT2523 will send the firmware to MT5932 by SDIO,
         then MT5932 will boot up.
    - There are three buttons on the board:
       - RST - reset.
       - EINT - external interrupt trigger.
       - RTC_INT - RTC interrupt trigger.

@par Run the demo
  - Connect the board to a PC with a USB cable.
  - Build this example project, download the two binary files in "<sdk_root>/out/mt2523_hdk/wifi5932_ref_design" folder to the LinkIt
    2523 HDK.
  - Reboot the 2523 HDK, it starts download the image to 5932 HDK.
  - When download complete, then 5932 HDK will boot up.
  - AT commands are supported for the demo.

@par Supported AT commands
  - AT+WIFISTA=scan,<max_ap_num>
    <max_ap_num>: The number of scan table wants to scan, the MAX number is 100
    Example:
    AT+WIFISTA=scan,30            : Start scan, and the number of scan table is 30.
    
  - AT+WIFISTA=scanmode,<start/stop>,<active/passive>,<partial/full>
    <start/stop>: 1: Start scan; 0: Stop scan.
    <active/passive>: 1: Passive scan; 0: Active scan.
    <partial/full>: 1: Partial scan(not support); 0: Full scan.
    Example:
    AT+WIFISTA=scanmode,1,0,0      : Start scan with full and active scan
    
  - AT+WIFI=<start/stop>
    <start/stop>: 1: Start Wi-Fi; 2: Stop Wi-Fi.
    Example:
    AT+WIFI=1                      : Enable Wi-Fi of MT5932
    AT+WIFI=0                      : Disable Wi-Fi of MT5932
