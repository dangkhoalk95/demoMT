Overview

This section introduces the advertising Bearer layer and GATT Bearer layer APIs.

It shows on how to use the enums and structures and the API prototypes to set the advertising parameters and scan parameters for the advertising Bearer layer, and how to connect and disconnect with the GATT Bearer layer.

Setting up Advertising bearer parameters.

Parameters
[in]adv_paramsis the advertising parameters as defined in bt_mesh_bearer_adv_params_t.
[in]scan_paramsis the scanner parameters as defined in bt_mesh_bearer_scan_params_t.

BT_MESH_SUCCESS, means the advertising parameters were successfully set.
