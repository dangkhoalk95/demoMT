Introduction

Overview

MT5931 is an early Wi-Fi only chip produced by Mediatek. It supports features including 11a/11b/11g/11n/QoS/WEP/WPA/WPA2/WAPI/WPS/P2P/WFD. It uses SDIO or EHPI interface to communicate with the host platform. It was widely used on the platforms of Feature Phone/MID/Car TV. Now it is also used on IoT platform after removing unused features according to the requirement of IoT market.
