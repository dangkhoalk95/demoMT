Overview

This section introduces the mesh configuration APIs.

It shows how to: use enums and structures, and API prototypes to enable and disable features; set and get the default TTL; add and delete the network key, application key, or device key; update the network key and application key; set the replay protection size or sequence number or iv index; and how to get other mesh network parameters.
