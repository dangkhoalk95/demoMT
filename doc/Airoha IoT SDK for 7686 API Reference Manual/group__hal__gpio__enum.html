enum hal_gpio_clock_mode_t

This enum defines output clock mode of GPIO.

Enumerator
HAL_GPIO_CLOCK_MODE_32K 

32KHz

HAL_GPIO_CLOCK_MODE_FXO 

if XO clock is 40MHz, output 20MHz.

If XO is 26MHz, output 26MHz

HAL_GPIO_CLOCK_MODE_24M 

24MHz,divided from BBPLL2 (960MHz/40 = 24MHz)

HAL_GPIO_CLOCK_MODE_48M 

48MHz,divided from BBPLL2 (960MHz/20 = 48MHz)

HAL_GPIO_CLOCK_MODE_2M 

2MHz, divided from fxo (20MHz/10 = 2MHz or 26MHz/13 = 2MHz)

HAL_GPIO_CLOCK_MODE_40M 

40MHz,divided from BBPLL2 (960MHz/24 = 40MHz)

HAL_GPIO_CLOCK_MODE_XPLL 

24.576MHz or 22.5792MHz (by I2S required setting)

HAL_GPIO_CLOCK_MODE_XO_BBTOP 

40Mhz or 26Mhz, it's depend on XO clock

HAL_GPIO_CLOCK_MODE_MAX 

define GPIO output clock mode of max number(invalid)
